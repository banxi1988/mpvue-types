# mpvue-types 
使用 mpvue 开发微信小程序的一些辅助的类型声明

# Install

```
npm i mpvue-types --save-dev
```

# How to Use

## add this module in your tsconfig.json

```json
{
	"compilerOptions": {
		"types": [
			"mpvue-types"
		]
	}
}
```

## add implements to your mpvue page component 

then you can get doc tip and autocomplete
